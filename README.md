# Commanders Expected Value Analysis

## Overview
Analytics assessment completed for the Washington Commanders R&D 
internship program analyzing expected value and play-by-play 
performance using NFL data.

## Tools Used
- Python
- Pandas
- Matplotlib
- nflfastR

## Key Findings
- Calculated expected value of plays using NFL play-by-play data
- Built visualizations using Matplotlib to present findings
- Analyzed offensive efficiency metrics across game situations
